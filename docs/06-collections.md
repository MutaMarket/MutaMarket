# Collections

## What are collections?

Collections help you organize and track modules by:

- Creating custom categories for your modules
- Tracking modules you're interested in
- Organizing modules by build or purpose
- Sharing collections with other users

## How do I create a collection?

1. Go to the [Collections page](/collections)
2. Click "Create Collection"
3. Give your collection a name and description
4. Choose your collection's visibility:
   - **Private**: Only you can see and edit the collection
   - **Public**: Anyone can view the collection, but only you can edit it
   - **Unlisted**: Accessible to anyone with the direct link
5. Add modules to your collection

> **Note:** You can change the visibility settings of your collection at any time.

## What are the different collection visibility modes?

Collections can be set to three different visibility modes:

### Private Collections

- Only visible to you
- Perfect for personal module tracking
- Can be used to organize your own modules
- Others cannot see or search for the collection

### Public Collections

- Visible to all MutaMarket users
- Great for sharing module builds or recommendations
- Can be found through search
- Others can view but cannot edit

### Unlisted Collections

- Accessible to anyone with the direct link
- Not visible in public searches
- Perfect for sharing with specific people via link

> **Tip:** Use different visibility modes for different purposes - private for personal tracking, public for sharing
> builds, and unlisted for sharing via direct links.

## How do I manage my collections?

You can manage your collections by:

- Adding or removing modules
- Editing collection details
- Changing privacy settings
- Sharing collections with other users
- Adding notes to modules in your collection
