# About MutaMarket

Hi there—I'm Nicolas Kion, the developer behind MutaMarket. I built this platform to help fellow capsuleers take
control of their abyssal modules in EVE Online. MutaMarket isn't just another tool; it's a personal project born from
my own experience with messy in-game inventories and the challenge of tracking down that perfect module roll.

## My Journey & Vision

I created MutaMarket to solve a problem I faced daily: managing countless abyssal modules across multiple characters.
I envisioned a platform that would make module management both powerful and intuitive—one that combines advanced
features with a clean, visual interface. Every feature, from seamless ESI imports to advanced appraisal and
probability calculations, is designed with the user in mind.

## What MutaMarket Offers

### Module Management

- Easy module imports via EVE's API or chat messages
- Clean, intuitive interface
- Advanced search and filtering capabilities
- Flexible collection organization

### Trading Features

- Streamlined module listing process
- Seamless in-game contract integration
- Intuitive offer system for negotiations
- Easy collection sharing with others

## Technology & Innovation

MutaMarket is built with modern technologies to deliver a smooth, reliable experience:

- Machine learning algorithms for precise module valuations
- Real-time market data integration through EVE ESI
- Robust Laravel backend for reliable API handling
- Responsive Vue.js frontend for smooth interactions
- Modern Tailwind CSS for consistent design
